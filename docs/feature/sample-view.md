# 시료 관리 조회 (requirements.pdf p.13)

## 개요

등록된 시료(`Sample`) 전체 목록과 각 시료의 현재 재고 수량을 조회하는 화면.
요구사항 문서의 "시료 관리 - 예시 UI 화면"(p.13)에 대응한다.

## 동작

- `data-source.md`의 `SampleDataSource`로 시료 목록을 조회 전 매번 `reload()`한다
  (실시간성, `live-refresh.md` 참고).
- 목록을 ID/이름/평균 생산시간/수율/현재 재고 순서의 표로 출력한다(p.13 예시 UI 참고,
  화면 레이아웃 자체는 자유).
- 등록된 시료가 없으면 빈 목록임을 알린다(예외를 던지지 않는다).

## 관련 기능

- `data-source.md`: 시료 목록을 불러올 때 사용.
- `monitoring-summary.md`: 이 화면의 재고 수량 데이터를 재사용해 여유/부족/고갈 상태로
  요약한다.
